# Proceso sobre resultado

Un buen trade puede perder y un mal trade puede ganar. Lo que se controla es el
**proceso**, no el resultado de una operación aislada. El edge aparece en la serie
larga, solo si cada entrada cumple los criterios.

**Principios:**
- **Solo setups A+.** Si la entrada no cumple todas las condiciones del playbook,
  no se opera. "Casi" no es señal.
- **Confluencia, no señal única.** Una sola condición (ej. RSI sobreventa) no basta.
  Se necesitan 2-3 factores alineados (tendencia + nivel + patrón/momentum).
- **No forzar.** Si no hay setup claro, la mejor operación es no operar. El mercado
  siempre da otra oportunidad.
- **Consistencia.** Mismo criterio siempre; no cambiar las reglas por FOMO o revancha.

**Regla para el bot:** Devuelve "none" sin dudar cuando no hay confluencia clara (≥2 factores alineados). No inventes entradas para "tener actividad". Calidad sobre cantidad.
