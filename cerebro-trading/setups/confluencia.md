# Confluencia (apilar factores)

La confluencia es cuando **varios factores independientes apuntan a lo mismo** en el mismo
punto. Es lo que separa una entrada A+ de una corazonada. Más confluencia = mayor
probabilidad y mayor confianza.

**Factores que se apilan (ejemplo para LONG):**
1. **Estructura:** zona de demanda / MSB alcista / tercer toque (RektProof).
2. **Nivel:** soporte de timeframe mayor / número redondo / VaL del perfil.
3. **Tendencia:** a favor del timeframe alto ([[multi-timeframe]]).
4. **Momentum:** divergencia RSI alcista / MACD girando.
5. **Sentimiento:** miedo extremo (contrarian).
6. **Ballenas:** acumulación / salidas de exchange.
7. **Confirmación de vela:** pin bar / engulfing en la zona.

**Cómo decidir:**
- **1 factor = no operar.** **2–3 factores alineados = válido.** **4+ = alta convicción**
  (mayor confianza, puedes pesar más el trade dentro del riesgo del plan).
- Si los factores se **contradicen**, baja la confianza o "none". La confluencia debe ser real,
  no forzada.

**Regla para el bot:** Cuenta los factores de confluencia alineados antes de decidir. <2 = none. 2–3 = entrada válida con confianza media. 4+ = alta convicción. Factores en conflicto = baja la confianza. La confianza reportada debe reflejar cuántos factores se apilan.
